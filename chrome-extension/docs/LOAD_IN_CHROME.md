## Troubleshooting:

### Extension won't load?
**Error**: "Manifest file is missing or unreadable"
- **Fix**: Make sure you selected the `dist` folder, not `chrome-extension` folder

### Extension loads but shows errors?
**Error**: "Service worker registration failed"
- **Fix**: Check browser console (F12) for specific errors
- **Fix**: Try rebuilding: `npm run build`

### Button doesn't appear on Blackboard?
**Check**:
1. Are you on the correct URL pattern?
   - Must match: `luminate.centennialcollege.ca/ultra/courses/*/outline*`
2. Is the extension enabled?
   - Check `chrome://extensions/` - toggle should be blue
3. Did content script load?
   - Open DevTools (F12) â†’ Console
   - Look for errors related to content.js

### Popup is blank?
**Check**:
1. Open DevTools on the popup:
   - Right-click extension icon â†’ "Inspect popup"
2. Look for errors in Console
3. Common issues:
   - Missing permissions
   - CORS errors
   - React not loading

### Backend connection fails?
**Error**: "Backend not running" or timeout errors
**Fix**: 
1. Start the backend:
   ```bash
   cd development/backend/fastapi_service
   source ../../../.venv/bin/activate
   uvicorn main:app --reload
   ```
2. Verify health:
   ```bash
   curl http://localhost:8000/health
   ```
